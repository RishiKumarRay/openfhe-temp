template<typename Element>
class lbcrypto::CryptoContextFactory< Element >

CryptoContextFactory.

A class that contains static methods to generate new crypto contexts from user parameters

Construct a PALISADE CryptoContextImpl for the BGVrns Scheme.

Parameters
multiplicativeDepththe depth of multiplications supported by the scheme (equal to number of towers - 1)
ptmthe plaintext modulus
stdLevelthe standard security level we want the scheme to satisfy
stdDevsigma - distribution parameter for error distribution
maxDepththe maximum power of secret key for which the relinearization key is generated
modeRLWE (gaussian distribution) or OPTIMIZED (ternary distribution)
ksTechkey switching technique to use (e.g., HYBRID, GHS or BV)
ringDimthe ring dimension (if not specified selected automatically based on stdLevel)
numLargeDigitsthe number of big digits to use in HYBRID key switching
firstModSizethe bit-length of the first modulus
dcrtrBitsthe size of the moduli in bits
relinWindowthe relinearization windows (used in BV key switching, use 0 for RNS decomposition)
batchSizethe number of slots being used in the ciphertext
msMethodmod switch method
Returns
new context
